Use Contact Form 7 as a trigger to any webhook like readymode!

== Description ==

[Contact Form 7 (CF7)](https://wordpress.org/plugins/contact-form-7/ "Install it first, of course") is a awesome plugin used by 1+ million WordPress websites.

Webhooks are endpoint (urls) you can send data!

Now you can join both: the best contact form plugin to WordPress and a webhook to readymode!

[Readymode (Readymode)](https://readymode.com/) is the smartest, most powerful predictive auto dialer for call centers.

Just activate and configure Readymode to receive data!

Disclaimer: this plugin was created without any encouragement from Readymode or CF7 developers and any webhook/API service.

= How to Use =

Easily and quickly! Just activate "Contact Form 7" and "CF7 to Webhook" and configure a URL to send data (or go to Readymode).

= Creating your workflow =

After configuration you can send one form to create a example data into Readymode lead. Then you can continue creating your workflow with filters and other apps.

= Support =

* Contact Form 7 - 5.7.6

== Installation ==

`Install [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) and activate it.`

* Install "CF7 to Webhook" by plugins dashboard.

Or

* Upload the entire `cf7-to-readymode` folder to the `/wp-content/plugins/` directory.

Then

* Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Readymode' tab into form configuration.
